# Project Task Summary

This repository records my additional research training on compact objects and gravitational waves under the guidance of Shenghua Yu (NAOC).

The project focuses on simulated LISA data for an NS-WD binary system and aims to build a basic gravitational-wave signal analysis workflow, including:

1. Time-domain inspection of simulated data and templates
2. FFT-based frequency-spectrum analysis
3. STFT / spectrogram analysis for time-frequency evolution
4. Noise PSD estimation using Welch methods
5. Noise-weighted matched filtering
6. Peak marking and edge-effect diagnosis
7. Monte Carlo noise-only significance tests
8. Injection tests for recovery capability
9. Template-start scanning and refinement

This repository is not intended as a full research release, but as a structured record of the tasks I completed during this training stage.
